# creditcard

Wsdl Path: 
```sh
$ src/main/resources/creditcard.wsdl
```

Para correr el proyecto dockerizado:
```sh
$ ./startup.sh
```

Para eliminar el proyecto dockerizado:
```sh
$ ./stop.sh
```
